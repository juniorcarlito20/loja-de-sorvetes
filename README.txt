sistema para  Loja de Açaí e Cupuaçu — Sistema em Python.

Este projeto é um sistema simples em Python para simular pedidos em uma loja que vende Açaí e Cupuaçu, com diferentes tamanhos, validações de entrada e cálculo automático do valor total.

Foi desenvolvido como exercício acadêmico para praticar:
- Estruturas condicionais
- Laços de repetição
- Validação de dados
- Acumuladores
- Entrada e saída de dados no console


 Funcionalidades

- Exibe mensagem de boas-vindas personalizada
- Mostra cardápio com preços
- Valida sabor (CP ou AC)
- Valida tamanho (P, M ou G)
- Permite múltiplos pedidos
- Soma automaticamente o valor total
- Exibe o total final ao usuário
